## Installation

![Repo Size](https://img.shields.io/github/repo-size/exikoz/banken?style=for-the-badge&color=purple)


Follow these steps to download and run the project locally:


### 1. Choose a Folder
Pick or create a folder where you want to store the project files.


### 2. Open Command Prompt in That Folder
- Click the **address bar** in your File Explorer.
- Type `cmd` and press **Enter**.  
  This opens Command Prompt in the same directory.

### 3. Clone the Repository
Run the following command:

```bash
git clone https://github.com/exikoz/banken.git
```

## Features
###  Core functionalities
- Create and manage users and accounts.
- Deposit, withdraw, and transfer funds between accounts.
- Simulate currency conversions with exchange rates
- Persistent JSON storage for exchange rates.  
- Administrative actions
